Discover Isola

UX Experience / Graphic Design - Interactive Exhibition Locator for Milan's Isola Design District

Timeline

Nov '23 - Dec '23

Role

UX/UI Designer

Client

Fuorisalone Design Festival

Team

Collaborative Design Project

Project Overview

In this collaborative design project, I specialized in developing the interactive elements for the Isola Design District microsite. Our client was the Fuorisalone Design Festival, and our mission was to create an engaging digital platform that would enhance the exploration of Milan's vibrant Isola Design District while promoting local tourism and art exhibitions.

Project Goals

  1. Create an intuitive wayfinding system for district visitors
  2. Showcase local art exhibitions and venues
  3. Provide an engaging user experience that encourages exploration
  4. Integrate practical navigation with interactive elements
  5. Represent the district's vibrant creative culture through both print and digital design
  6. Ensure seamless transition between poster and microsite experiences

Integrated Design Approach

Interactive Microsite

The project centred on creating an intuitive interface where users could:

  • Navigate through the district using an interactive guide map
  • Discover art exhibitions and local attractions
  • Experience the district's atmosphere through preview videos
  • Calculate distances between locations
  • Access information about venues and events

Poster Design

  • Visual Sync: Sliding bars mirroring microsite loading
  • Color Palette: Pink, purple, lime green
  • Layout: Dynamic grid with district imagery
  • Interactivity: Static-to-digital transition
  • Typography: Suisse Int'l Bold, IBM Plex Mono
  • Consistency: Unified colors, geometric elements
  • Engagement: Print-to-digital visual flow

Design System

Integration Strategy

The design system was carefully crafted to ensure that:

  • Visual elements from the poster could be animated in the digital space
  • Color schemes remained vibrant and engaging across both mediums
  • Maintain readability while supporting the design aesthetic
  • Grid layouts could flex between static and dynamic presentations

Tools & Technologies

Design & Prototyping

Figma for prototyping and poster design

Animation

After Effects for smooth transitions

Mapping

Geographic Information Systems for journey mapping

Challenges

Visual Integration

  • Finding the perfect imagery that resonates with both Isola and Milan's character
  • Creating a cohesive visual language that works across print and digital formats
  • Maintaining text contrast and readability against vibrant color schemes

Technical Implementation

  • Developing smooth transitions between different interactive states
  • Integrating the guide map functionality with location previews
  • Ensuring consistent performance across different devices and screen sizes

Design Balance

  • Harmonizing the vibrant color palette across all elements
  • Creating engaging interactions without overwhelming users
  • Maintaining visual hierarchy while incorporating dynamic elements
