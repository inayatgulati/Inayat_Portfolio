The Woodland Vortex of Poh

Game Design: A 2D Tile-Based Adventure

Timeline

Jul '24 - Aug '24
(2 weeks)

Contributors

Solo Project

Role

Lead designer and developer, responsible for gameplay mechanics, user interface, and player interaction systems

Project Purpose

Created as a school project to develop skills in interactive design and game development

Project Overview

This project is a 2D tile-based adventure game where the player controls Poh, a panda, navigating through a map to collect items, defeat enemies, and unlock passages. The goal is to reach a winning tile, involving various interactions that test the player's strategy and decision-making. The game blends visual aesthetics with gameplay mechanics to create an engaging experience.

Context, Skills, and Tools

Skills

Game design, coding in JavaScript (P5.js), 2D graphics, UI/UX design for games

Tools

Figma for prototyping, Photoshop for sprite creation, and pictures, sounds taken from internet

Main Character

Meet Poh

The player controls Poh, a brave panda navigating through a mysterious woodland vortex, collecting items and overcoming obstacles to reach the gate exit.

Objective

The primary objective was to create a compelling, interactive experience that engages players through exploration and problem-solving within a 2D environment. The game addresses the challenge of designing mechanics that enhance user engagement and immersion while maintaining simplicity in navigation and controls.

Description of Solution

Goals

To develop an accessible and enjoyable gameplay experience for users of all ages, focusing on intuitive interaction and satisfying progression.

Design Rationale

Industry Standards & UI Patterns

  • Tile-based systems are a staple in 2D platformers because they provide a structured, modular environment that simplifies movement and navigation
  • Successful games like Super Mario Bros., Celeste, and Hollow Knight use tile-based layouts to create predictable yet dynamic gameplay spaces
  • In UI/UX design, grid-based layouts improve visual hierarchy, readability, and user efficiency, making them a trusted industry standard for organizing information clearly

Usability Research & Player Experience

  • Cognitive Load Reduction: Research in game usability suggests that structured layouts help users process information more efficiently, reducing confusion
  • Fitts's Law: A well-structured tile system groups interactive elements logically, making it easier for players to navigate and interact quickly
  • Nielsen Norman Group's UI Research highlights that grid-based structures guide user attention, improving scanability and interaction speed

Iterations

First Iteration: Basic movement and tile interaction were implemented, with feedback focused on optimizing controls.

Second Iteration: Added collectible items and enemy interactions, incorporating feedback to improve game balance and user feedback mechanisms.

Challenges and Discoveries

Initial challenges involved adjusting enemy AI to maintain fairness and enhancing visual elements for clarity. The design was adapted to balance difficulty and player progression.

Final Outcome

The final solution includes a complete gameplay loop with objectives, challenges, and a satisfying end goal, providing a cohesive and engaging user experience.

Challenges

Visual Integration

  • Finding the perfect imagery that resonates with the game theme
  • Creating a cohesive visual language across all game elements
  • Maintaining readability against dynamic backgrounds

Technical Implementation

  • Developing smooth transitions between different game states
  • Integrating collision detection with tile-based movement
  • Ensuring consistent performance across devices

Design Balance

  • Harmonizing difficulty progression throughout levels
  • Creating engaging interactions without overwhelming players
  • Maintaining visual hierarchy while incorporating dynamic elements

Takeaways

Learning

Gained insights into game mechanics and the balance between challenge and accessibility. Understanding how tile-based systems create predictable yet engaging gameplay experiences.

Future Improvements

Implementing more complex enemy AI and additional levels could enhance replayability and depth. Better optimization techniques for smoother performance across devices would also be considered.
